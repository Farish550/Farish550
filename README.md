# EasyFood - Food Discovery Platform

EasyFood is a web-based food discovery platform that helps tourists and locals find where to eat easily when they travel between cities or states.

## Project Overview

This is NOT a food delivery app and NOT an ordering system. The platform focuses on discovering restaurants, viewing menus, and navigating to the location.

## Core Features

- **Search & Filter**: Find restaurants by location, cuisine type, price range, and dietary preferences
- **Restaurant Listings**: Browse restaurants with essential information
- **Detailed Restaurant Pages**: View descriptions, photos, menus, opening hours, and directions
- **Responsive Design**: Works on both desktop and mobile devices
- **User-Friendly Interface**: Clean, modern design focused on usability

## Pages Included

1. **Home Page** - Search and featured restaurants
2. **Restaurant Listings** - Filtered restaurant browsing
3. **Restaurant Detail** - Comprehensive restaurant information
4. **About Page** - Information about EasyFood
5. **Contact Page** - Contact form and social links

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Responsive design with Flexbox and Grid
- Google Fonts (Poppins)
- Google Maps embedding

## How to Run

1. Open any HTML file in a web browser
2. No server required for static files
3. All functionality works in the browser
