How to Use:

  • All HTML files are located in the /src directory
  • Open any HTML file directly in your browser to view the page
  • Run the development server with: ./start_server.sh (requires Python)
  • Styles are in /src/css/style.css
  • JavaScript is in /src/js/main.js
